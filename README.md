# Tap-it

This App is Very Simple Game to play
Of the 3*3 Grid Of the Grid is Blink red color ab Tap it.


## Usage

- UITapGesture
- UserDefaults
- Tableview
- UIview
- UIbutton
- UIlabel
- etc..
